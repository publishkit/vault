# bg


> Customise the background

## use

```yaml
plugins:
  bg: true
```

## options

```yaml
bg:
  url: https://images.unsplash.com/photo-1539035104074-dee66086b5e3?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjI0MX0&auto=format&fit=crop&w=2550&q=80
  parralax: true
```
