# Community

The community themes & plugins are hosted on github at https://github.com/publishkit/community, and can be invoked from the frontmatter with the `@` symbol. Ex:

```yaml
# @file=kitc.md
plugins:
  theme: "@matrix"
  ga: "@ga"
```
