# fonts

> This plugin let you customize your site fonts.

You can choose between 19,723 font families from [cdnfonts.com](https://www.cdnfonts.com/)

## use

```yaml
plugins:
  fonts: true
```

## options

```yaml
fonts:
  font: Marcher
  headings: Bungee Shade
```
